# interface-ifood
Atividade desenvolvida para a plataforma DIO, atividade "Criando a interface do Ifood com o React Native". Assim sendo, foi apresentada a interface inicial do aplicativo iFood, com a listagem de restaurantes, banners e categorias. Utilizando React Native, os Hooks do React para controle de estado, React Navigation para roteamento de telas na aplicação e integrando a uma API estática com apenas um JSON para listagem dos restaurantes no app e muito mais!
